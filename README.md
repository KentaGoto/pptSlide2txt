# pptSlide2txt
Extract text from Powerpoint slides

## USAGE
```
> pptSlide2txt.pl [-nfn]
```
**WARNINGS:** 
Strings outside the slide are not extracted.


## Command Line Syntax:
- `\-nfn` : Do not output file name separator

## REQUIREMENTS
- Windows
- PowerPoint
- Perl (5.26 or above)
